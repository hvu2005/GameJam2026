# Tool Scripts

## SimpleMapLoader

Load map prefabs bằng MapData ScriptableObject assets.

### Setup:
1. Tạo MapData assets (Right-click → Create → Game → Map Data)
2. Gán vào GameObject trong scene
3. Nhấn R để load map theo ID

### Usage:
- Đổi `Current Map Id` trong Inspector
- Press **R** to load map
